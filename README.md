# ☕ 1. Coffee Shop Sales Analysis

## 📌 Project Overview

A creative and dynamic sales analytics dashboard built using a retail coffee shop dataset sourced from Maven Analytics.
The goal of this project was to analyze sales performance, identify customer behavior patterns, and generate actionable business insights that support data-driven decision-making.

This project includes:

* Excel workbook with:

  * Raw datasets
  * Pivot tables
  * Interactive dashboard
* Dashboard image preview in PNG format

---

## 🎯 Business Objectives

The project answers key business questions such as:

* How do sales vary by day and hour?
* What are the peak sales periods?
* What is the monthly revenue trend?
* Which store locations perform best?
* What is the average order value per customer?
* Which products generate the highest revenue?
* How do product categories contribute to overall sales?

---
